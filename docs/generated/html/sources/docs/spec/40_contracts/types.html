3 ## 1. 用語集
4 - ProtocolPackage: CSV取込後に保存される手技テンプレート。
5 - Session: 当日スロットに紐づく実施インスタンス。
6 - Record: `record_event` 単位の記録。
7 - SessionProtocolSnapshot: 開始時固定の手順定義。
8 - OutboxMutation: 同期対象のローカル更新。
9

239 ## 4. 不変条件（Invariants）
240 1. `record` のLWW比較キーは `(updatedAt, updatedByDeviceId, mutationId)` 降順。
241 2. `session_summary.payload.exit_site_photo` の更新は `patch_path=payload.exit_site_photo` の部分パッチ。
242 3. `SessionProtocolSnapshot` 保存失敗時はセッション開始失敗。
243 4. `cloudState=missing` 判定時にローカルデータ削除を禁止。
244 5. Import失敗時はテンプレート保存を行わない。
