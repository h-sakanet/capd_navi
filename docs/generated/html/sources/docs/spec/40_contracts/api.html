1 # API Contract
2  
3 ## 1. エンドポイント一覧
4 公開HTTP APIは以下2つのみです。
5  
6 | Method | Path | 目的 | 主入力 | 主出力 |
7 |---|---|---|---|---|
8 | POST | `/sync/push` | outbox mutation反映 | `deviceId`, `syncMode`, `baseCloudRevision`, `mutations[]` | `acceptedMutations[]`, `newCloudRevision`, `reseedApplied` |
9 | POST | `/sync/pull` | revision以降差分取得 | `deviceId`, `knownCloudRevision`, `knownDayRevisions` | `cloudState`, `cloudRevision`, `manifestDiff`, `dayBundles[]`, `photoRefs[]` |
10  
11 ローカルI/F（公開HTTP API対象外）:
12 - `ProtocolImportService.importFromDirectory`（CSV取込）
13 - `RecordService.updateRecord(..., patch_path=payload.exit_site_photo)`
14  
15 ## 2. リクエスト/レスポンス
16 - 共通入力: `deviceId`, `request timestamp`
17 - 共通出力: `status相当`, `error情報`, `cloudRevision`
18 - `syncMode`: `delta | full_reseed`
19 - `cloudState`: `ok | missing`
20  
21 `cloudState=missing` 判定条件:
22 - `index.json` 欠損
23 - 必須フィールド欠落/不正
24 - `dayRefs` / `photoRefs` 参照先欠損
25 - `integrity` 検証失敗
26  
27 ### 2.1 入出力例（固定）
28 `POST /sync/push` 入力例:
29 ```json
30 {
31 "deviceId": "macbook-main",
32 "syncMode": "delta",
33 "baseCloudRevision": 100,
34 "mutations": [
35 {
36 "mutationId": "mut_01J...",
37 "entityType": "record",
38 "entityId": "rec_01H...",
39 "operation": "patch",
40 "patch_path": "payload.exit_site_photo",
41 "payload": {
42 "exit_site_photo": {
43 "photo_id": "pho_exit_01H...",
44 "captured_at": "2026-02-09T21:15:00+09:00",
45 "uploaded_at": "2026-02-09T21:16:00+09:00",
46 "source_device_id": "iphone-main"
47 }
48 }
49 }
50 ]
51 }
52 ```
53  
54 `POST /sync/push` 出力例:
55 ```json
56 {
57 "acceptedMutations": ["mut_01J..."],
58 "rejectedMutations": [],
59 "newCloudRevision": 102,
60 "reseedApplied": false
61 }
62 ```
63  
64 `POST /sync/pull` 入力例:
65 ```json
66 {
67 "deviceId": "iphone-main",
68 "knownCloudRevision": 100,
69 "knownDayRevisions": {
70 "2026-02-09": "sha256:old"
71 }
72 }
73 ```
74  
75 `POST /sync/pull` 出力例:
76 ```json
77 {
78 "cloudState": "ok",
79 "cloudRevision": 102,
80 "manifestDiff": {},
81 "dayBundles": [],
82 "photoRefs": [
83 {
84 "photoId": "pho_exit_01H...",
85 "blobKey": "photos/pho_exit_01H....jpg",
86 "photo_kind": "exit_site",
87 "sizeBytes": 202114,
88 "sha256": "7af1...",
89 "updatedAt": "2026-02-09T21:16:00+09:00"
90 }
91 ]
92 }
93 ```
94  
95 `cloudState=missing` の返却条件（いずれか該当で返却）:
96 - `index.json` が存在しない
97 - `index.json` の必須フィールド（`cloudRevision`, `dayRefs`, `photoRefs`, `tombstones`, `integrityHash`）が欠落または不正
98 - `index.json` が参照する `days/*.json` の必須オブジェクトが1件でも欠損
99 - `index.json` が参照する `photos/*` が1件でも欠損
100 - `integrityHash` 検証が失敗し、整合性を回復できない
101  
102 ## 3. エラーコード
103 - UI表示コードは `./error-codes.md` を参照。
104 - API内の詳細失敗理由は `SyncState.lastError` に集約します。
