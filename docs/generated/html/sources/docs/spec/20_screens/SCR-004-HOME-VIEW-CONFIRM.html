10 ## 2. 目的と責務境界
11 - この画面で決めること: 閲覧専用の手順確認導線。
12 - この画面で決めないこと: 記録保存、タイマー生成、通知生成、スロット/セッション状態更新。

16 - 注意文「確認モード。保存なし」を固定表示します。
17 - ボタンは `[閉じる] [手順を表示（保存なし）]` を固定。

37 ## 7. バリデーション / エラー文言 / 空状態
